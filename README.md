# flyer-cli

> 商业端脚手架-飞鸟

## install

> npm install flyer-cli -g

# API

> ###脚手架包含了⑦种命令，分别是:
> * add: 添加模板，没有任何特性的模板
> * dellete: 删除模板
> * init: 初始化项目
> * list: 初始化一个列表模板
> * form: 初始化一个表单模板
> * report: 初始化一个报表模板
> * module: 初始化一个模块，包含 < 列表页(list.vue) | 表单页(form.vue) | 模块配置文件(config.js) >
